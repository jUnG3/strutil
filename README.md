# strutil
String util for C. Provided functionality:
* `strexplode`
* `trim`
* `strexplode_free`

# Test
Run the configured unit tests.
```bash
make test
```

# Build
```bash
make all
```

# Install
```bash
sudo make install
```
| OS | Header | Lib |
| --- | --- | --- |
| Linux | `/usr/local/include/strutil.h` | `/usr/local/lib/libstrutil.so` |
| OSX | `/usr/local/include/strutil.h` | `/usr/local/lib/libstrutil.dylib` |
| Windows | `` | `` |
